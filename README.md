# JLang

JLang is a lightweight, beginner-friendly scripting language built for an in-browser IDE.  
It’s designed to be **simple, readable, and interactive**, with built-in GUI, mini-games, and real-time execution.

---

## Features

- 🧠 Easy-to-learn custom syntax
- 🖥️ Web-based IDE (HTML + JS)
- 🖥️ Application IDE (Python Based)
- 🎮 Built-in 2D mini game system
- 🎛️ GUI elements (buttons, text, titles)
- ⚡ Real-time interpreter (no compilation needed)
- 🎨 Live styling (colors, backgrounds, UI switching)
- 🔁 Loop/repeat system
- ⏱️ Wait/delay execution
- 🔊 Sound support (beeps)
- 📦 Variable system
- 🧩 Extensible command dictionary
